universal-tilt.js

With universal-tilt.js You can add movement to elements on your website (or app).
The script is based on the position of the cursor over the item (desktop) or on the gyroscope movement (mobile devices).

You can use universal-tilt.js with Vanilla JavaScript e.g:

var tilts = document.querySelectorAll('.tilt');

var options = {};

var liveTilt = new UniversalTilt(tilts, options);

or jQuery:

$('.tilt').UniversalTilt();

Shine

To add shine to single element use following code in div tag:

data-shine-opacity="0.3"

To add shine to all elements place in JavaScript (or jQuery) options:

shine: true,

'shine-opacity': 0.3,
'shine-save': true

Shadow

To add shadow to single element use following code in div tag:

data-shadow="true"

To add shadow to all elements place in JavaScript (or jQuery) options:

shadow: true,
'shadow-save': true

Scale

To add scale to single element use following code in div tag:

data-scale="1.3"

To add scale to all elements place in JavaScript (or jQuery) options:

scale: 1.3

Parallax

Add to your tilt element:

transform-style: preserve-3d

Add to your inner elements that have to pop out:

transform: translateZ(20px)

Disable X

To disable X-axis of single element use:

data-disabled="x"

To disable X-axis of all elements place in JavaScript (or jQuery) options:

disabled: 'X'

Disable Y

To disable Y-axis of single element use:

data-disabled="y"

To disable Y-axis of all elements place in JavaScript (or jQuery) options:

disabled: 'Y'

Reset (Only desktop)

To disable reset single element move on mouse out use:

data-reset="false"

To disable reset all elements move place in JavaScript (or jQuery) options:

reset: false

Reverse

To dreverse single element move use:

data-reverse="true"

To reverse all elements move place in JavaScript (or jQuery) options:

reverse: true

Position base (Only desktop)

Tilt base on cursor position over tilt element or window

data-position-base="window"

Tilt base on cursor position over tilt element or window

'position-base': 'window'

Change event

Change event will output the x, y & percentages of tilting.
