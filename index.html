Usage

At the beginning connect the library with Your project:

• using script tag in HTML:

<script src="/directory/to/library/folder/universal-tilt.js"></script>

• or via command line and CommonJS/ES6 import:

npm install universal-tilt.js // npm

yarn add universal-tilt.js // yarn
const UniversalTilt = require('universal-tilt.js'); // CommonJS

import UniversalTilt form 'universal-tilt.js'; // ES6

Next use library with:

• Vanilla JavaScript e.g:

const tilts = document.querySelectorAll('.tilt');

const liveTilt = new UniversalTilt(tilts, {
  // options...
});

• or jQuery e.g:

Connect jQuery in HTML 
<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>

or include via command line and CommonJS 
npm install jquery // npm

yarn add jquery // yarn

bower install jquery // bower
const jQuery = require('jquery');

and call plugin on element 
$('.tilt').universalTilt({
  // options...
});

Shine

To add shine to single element use following code in div tag:

data-shine="true"
data-shine-opacity="0.5"
data-shine-save="true"

To add shine to all elements place in JavaScript (or jQuery) options:

shine: true,

'shine-opacity': 0.5,
'shine-save': true

Shadow

To add shadow to single element use following code in div tag:

data-shadow="true"
data-shadow-save="true"
data-shadow-color="rgba(255, 255, 255, 0.2)"

To add shadow to all elements place in JavaScript (or jQuery) options:

shadow: true,
'shadow-save': true,
'shadow-color': 'rgba(255, 255, 255, 0.2)'

Scale

To add scale to single element use following code in div tag:

data-scale="1.3"

To add scale to all elements place in JavaScript (or jQuery) options:

scale: 1.3

Parallax

Add to your tilt element:

transform-style: preserve-3d

Add to your inner elements that have to pop out:

transform: translateZ(20px)

Disable X

To disable X-axis of single element use:

data-disabled="x"

To disable X-axis of all elements place in JavaScript (or jQuery) options:

disabled: 'X'

Disable Y

To disable Y-axis of single element use:

data-disabled="y"

To disable Y-axis of all elements place in JavaScript (or jQuery) options:

disabled: 'Y'

Reset (Only desktop)

To disable reset single element move on mouse out use:

data-reset="false"

To disable reset all elements move place in JavaScript (or jQuery) options:

reset: false

Reverse

To reverse single element move use:

data-reverse="true"

To reverse all elements move place in JavaScript (or jQuery) options:

reverse: true

Position base (Only desktop)

Tilt base on cursor position over tilt element or window

data-position-base="window"

Tilt base on cursor position over tilt element or window

'position-base': 'window'

Change event

Change event will output the x, y & percentages of tilting.
